# RS485 从站 1

从站 1 负责光照采集、阈值判断和继电器控制，所有状态通过 Modbus 寄存器对外提供。

## 角色定位

- 采集 ADC 光照值。
- 依据阈值控制继电器输出。
- 维护寄存器映射，供主站读取和写入。

## 关键模块

- `App/Src/light_control.c`：光照采集与继电器控制。
- `App/Src/rs485_slave.c`：Modbus 从站协议处理。
- `App/Src/rs485_driver.c`：RS485 收发驱动。
- `Core/Src/adc.c`：ADC 外设初始化。

## 寄存器映射

| 地址 | 类型 | 说明 |
|---|---|---|
| 0x0000 | 只读 | ADC 原始值 |
| 0x0001 | 读写 | 光照阈值 |
| 0x0002 | 只读 | 当前照度值 |
| 0x0003 | 只读 | 继电器状态 |

## 对外行为

- 主站可读取照度和继电器状态。
- 主站可写入光照阈值。
