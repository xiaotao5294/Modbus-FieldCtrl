# RS485 主站

主站工程，负责轮询两个从站、解析控制台命令并输出诊断信息。

## 角色定位

- 以 FreeRTOS 组织 RS485 通信状态机。
- 通过 USART2 接收控制台命令。
- 轮询从站并维护在线、超时和诊断信息。
- 对从站 2 下发舵机角度等控制量。

## 关键模块

- `Core/`：系统启动、中断和外设初始化。
- `App/Src/task_rs485.c`：RS485 任务与轮询调度。
- `App/Src/rs485_protocol.c`：Modbus 帧处理。
- `App/Src/rs485_console.c`：串口命令解析。
- `App/Src/rs485_scheduler.c`：轮询节拍与任务编排。

## 工作流程

1. 控制台收到命令后，转换为内部请求。
2. RS485 任务按周期发起轮询或写入。
3. 从站响应后进入解析、校验和状态更新。
4. 结果通过控制台或诊断日志输出。

## 对外接口

- 读取从站状态和寄存器快照。
- 写入从站 1 阈值参数。
- 写入从站 2 舵机角度。
