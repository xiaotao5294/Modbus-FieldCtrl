# RS485 从站 2

从站 2 负责 MPU6050 采集和舵机控制，主站通过写寄存器直接设定舵机角度。

## 角色定位

- 采集 MPU6050 三轴原始数据。
- 将传感器数据和舵机角度映射到 Modbus 寄存器。
- 接收主站写入的角度并驱动 PWM 输出。

## 关键模块

- `App/Src/mpu6050_sensor.c`：MPU6050 数据采集。
- `App/Src/servo_control.c`：舵机 PWM 控制。
- `App/Src/rs485_slave.c`：Modbus 从站协议处理。
- `App/Src/rs485_driver.c`：RS485 收发驱动。

## 寄存器映射

| 地址 | 类型 | 说明 |
|---|---|---|
| 0x0000 | 只读 | MPU6050 X 轴原始值 |
| 0x0001 | 只读 | MPU6050 Y 轴原始值 |
| 0x0002 | 只读 | MPU6050 Z 轴原始值 |
| 0x0003 | 读写 | 舵机角度（0-180） |

## 对外行为

- 主站可读取三轴原始数据。
- 主站可写入舵机角度。
